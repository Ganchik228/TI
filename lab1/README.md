# Приложение для расчета энтропии и информации

## Описание

Это приложение Windows Forms для расчета вероятностей, энтропии и взаимной информации двух источников. Приложение поддерживает три различных случая ввода данных.

## Функциональность

### Поддерживаемые случаи расчета:

1. **Матрица совместных вероятностей P(AB)**

   - Ввод матрицы совместных вероятностей P(a_i, b_j)
   - Автоматический расчет всех остальных величин

2. **Матрица условных вероятностей P(A/B) и ансамбль B**

   - Ввод матрицы условных вероятностей P(A/B)
   - Ввод ансамбля B (вероятности b_j)
   - Автоматический расчет всех остальных величин

3. **Матрица условных вероятностей P(B/A) и ансамбль A**
   - Ввод матрицы условных вероятностей P(B/A)
   - Ввод ансамбля A (вероятности a_i)
   - Автоматический расчет всех остальных величин

### Рассчитываемые величины:

- **Ансамбль A и B** - маргинальные вероятности
- **Энтропия H(A)** - энтропия ансамбля A
- **Энтропия H(B)** - энтропия ансамбля B
- **Условная энтропия H(B|A)** - энтропия B при условии A
- **Условная энтропия H(A|B)** - энтропия A при условии B
- **Совместная энтропия H(A,B)** - энтропия совместного распределения
- **Взаимная информация I(A,B)** - мера взаимной зависимости

## Использование

### Ввод данных:

1. **Выберите случай** из выпадающего списка
2. **Установите размер матрицы** (количество строк и столбцов)
3. **Введите значения** в матрицу:
   - Поддерживаются как точки (0.1), так и запятые (0,1) как десятичные разделители
   - Для случаев 2 и 3 дополнительно введите ансамбли через запятую

### Примеры данных:

**Случай 1 (P(AB)):**

```
0,1   0,06  0,02  0,02
0,16  0,08  0,4   0,16
```

**Случай 2 (P(A/B) + ансамбль B):**

```
Матрица:
0,5  0,3  0,1  0,1
0,2  0,1  0,5  0,2

Ансамбль B: 0,26, 0,14, 0,42, 0,18
```

**Случай 3 (P(B/A) + ансамбль A):**

```
Матрица:
0,3846  0,4286  0,0476  0,1111
0,6154  0,5714  0,9524  0,8889

Ансамбль A: 0,2, 0,8
```

### Получение результатов:

1. Нажмите кнопку **"Рассчитать"**
2. Результаты отобразятся в правой части окна
3. Матрица совместных вероятностей отобразится в верхней таблице справа
4. Матрицы условных вероятностей P(A|B) и P(B|A) отобразятся в табах в нижней части окна

## Технические особенности

- Поддержка русского языка и локали (запятые как десятичные разделители)
- Проверка корректности введенных данных
- Автоматический расчет всех производных величин
- Удобный интерфейс с разделением на ввод и вывод

## Сборка и запуск

```bash
dotnet build
dotnet run
```

Приложение создано на платформе .NET 9.0 с использованием Windows Forms.
